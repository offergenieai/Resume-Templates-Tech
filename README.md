# Tech Resume Templates 2026

> ATS-friendly resume templates and examples for tech roles.

## Resume Resources

### Software Engineer Resume

A strong SWE resume should include:

- **Contact Information** — Name, email, LinkedIn, GitHub
- **Summary** — 2-3 sentences highlighting your experience and skills
- **Skills** — Programming languages, frameworks, tools, cloud platforms
- **Experience** — Bullet points with quantifiable achievements (STAR method)
- **Projects** — Personal/open-source projects with tech stack and impact
- **Education** — Degree, university, GPA (if > 3.5), relevant coursework

### Resume Optimization Tips

1. **Use ATS-friendly formatting** — Simple layout, no tables/columns, standard fonts
2. **Quantify achievements** — "Reduced API latency by 40%" > "Improved performance"
3. **Tailor to each role** — Match keywords from the job description
4. **Keep it concise** — 1 page for < 5 years experience, 2 pages max
